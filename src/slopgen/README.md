# slopgen package

**EN:** Top-level package. Subpackages: `cli` (Typer entrypoint), `tui` (Textual interface), `config` (TOML config models/loader), `llm` (DeepSeek client), `media` (ffmpeg + stock footage), `pipeline` (orchestrator and stages), `publish` (upload backends).

**RU:** Корневой пакет. Подпакеты: `cli` (входная точка Typer), `tui` (интерфейс Textual), `config` (модели/загрузчик TOML-конфигов), `llm` (клиент DeepSeek), `media` (ffmpeg + стоковый футаж), `pipeline` (оркестратор и стадии), `publish` (бэкенды загрузки).
