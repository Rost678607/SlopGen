# media

**EN:** `ffmpeg.py` — subprocess wrappers: silent background parts (`make_video_part` loops+crops clips, `make_photo_part` does Ken Burns via zoompan), `make_scene_segment` composes background parts + voiceover + framed foreground picture inserts, plus concat and the final composition (burned ASS subtitles, background music mix with loudnorm, ad overlay + caption). `stock.py` — stock providers for both videos (`find_clip`) and photos (`find_image`): Pexels, Pixabay, local `assets/footage` / `assets/images`, with an on-disk download cache in `state/cache/`. `generate.py` — free AI providers slotted into the same chains: `pollinations` (keyless text-to-image, photo sources) and `wan` (text-to-video via HF Spaces — Wan 2.1 with LTX/AnimateDiff reserves — slow, degrades to the next provider on failure). Enable per project by adding them to `[footage] providers` in `configs/slopgen.toml`.

**RU:** `ffmpeg.py` — subprocess-обвязка: немые фоновые части (`make_video_part` — луп+кроп клипов, `make_photo_part` — Ken Burns через zoompan), `make_scene_segment` собирает фон + озвучку + вставки-картинки в рамке, плюс конкатенация и финальная композиция (вжигание ASS-сабов, музыка с loudnorm, рекламный оверлей + подпись). `stock.py` — сток-провайдеры видео (`find_clip`) и фото (`find_image`): Pexels, Pixabay, локальные `assets/footage` / `assets/images`, дисковый кэш в `state/cache/`. `generate.py` — бесплатные ИИ-провайдеры в тех же цепочках: `pollinations` (text-to-image без ключа, для фото-источников) и `wan` (text-to-video через HF Spaces — Wan 2.1 с резервами LTX/AnimateDiff — медленно, при сбое переходит к следующему провайдеру). Включаются добавлением в `[footage] providers` в `configs/slopgen.toml`.
