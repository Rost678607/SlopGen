# cli

**EN:** Typer-based CLI. `slopgen` with no arguments launches the TUI; `slopgen <lang> <type> [flags]` runs headless. Parameter resolution order: CLI flags > preset > account defaults > global defaults. Also hosts the `--list-*` inspection flags and the console progress printer.

**RU:** CLI на Typer. `slopgen` без аргументов запускает TUI; `slopgen <язык> <тип> [флаги]` — headless-режим. Приоритет параметров: флаги CLI > пресет > дефолты аккаунта > глобальные дефолты. Здесь же флаги `--list-*` и консольный принтер прогресса.
